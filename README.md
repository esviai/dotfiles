# dotfiles
A very simple dotfiles from  someone who is still learning vim and tmux the hard way.

## .vimrc
**Based on Amir Salihefendic's dotfiles**

Plugins: 
- vim-javascript
- vim-jsx
- emmet-vim
- vim-json
- ale (linters: eslint)

Theme: solarized

## .tmux-conf

Functions:
- remap prefix from 'C-b' to 'C-a'
- vi mode copy paste

Theme: solarized
